# 게시판 개요

## 글 리스트

    - 글 쓰기 폼
    - 글 보기
        - ✔ 파라미터
            - 게시글 번호(PK)
            - 게시글이 있는 페이지 번호
            - [검색 컬럼, 검색 값]
    - 페이징 처리 → 글 리스트
        - ✔ 파라미터
            - 현재 페이지 번호
    - 검색 → 글 리스트
        - ✔ 파라미터
            - 검색 컬럼(제목, 글 작성자, 내용 등...)
            - 검색 값

## 글쓰기 폼

    - 글 쓰기 폼 → 글 등록 → 글 리스트
        - ✔ 파라미터
            - 제목
            - 내용
            - 작성자
            - 암호
    - 취소 → 글 리스트

# 경로 개념

## / == 최상위 경로

/: http://localhost:포트/

    - HTML, CSS, JAVASCRIPT, response.sendRedirect

## / == context path

/: http://localhost:포트/프로젝트명/

    - 이클립스의 webContent폴더의 경로
    - forwarding할 때의 경로
