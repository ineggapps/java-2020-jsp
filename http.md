# HTTP

## 상탯값 저장

- HTTP프로토콜은 기본적으로 비연결형이므로 실시간으로 끊임없이 주고받는 것이 아니다.
- 그런데 가끔은 상태 정보를 저장하고 있어야 할 때가 있다.

## [쿠키](https://ko.wikipedia.org/wiki/HTTP_%EC%BF%A0%ED%82%A4)

- 서버에서 만들어서 클라이언트에 상탯값을 저장하는 방식

## [세션](<https://ko.wikipedia.org/wiki/%EC%84%B8%EC%85%98_(%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B3%BC%ED%95%99)>)

- 서버 자체에 저장하는 방식.
- 주로 로그인 정보를 저장한다.
  - 그러나 모바일 장치에서는 세션이 아니라 쿠키에 저장한다는 것에 유의한다.
